 a cross platform Saas named zabunilink,
for Tanzania businesses where the can get the tender lists and they can either apply themselves or we do it for them.
the core idea is to help businessmen to get tender details, to get notification and do it earlier, also we help pro business to apply tender and do everything for them.


core features:
- Membership platform
- notification
- Membership payment fee collection

Requirements
- Web portals (user/member portal and admin portal)
- android app
- ios app

android and ios apps for users
so for user there is web app, android app and ios app
for admin just a web portal


Core engines / Getaways / Mechanism
-Membership engine
-Tender Engine
-Email Engine
-Sms Engine
-Push Notification Engine
-Payment Collection Engine


Membership
2 packages for businesses (basic and Pro)
Monthly Membership Account 
membership fee collected monthly, quartely, semi, or annually

basic packages is basically a portal that list all the tenders (government , private etc) with all their details 
Pro package give you ability to request application where by we can apply and comply for you and do all the heavy lifting and you just fulfill it.



in both packages you get notification access

Tender List
-Notification to members of new tender (sms, email, push)
-Tender list that list all tenders
-Tender Searching and Filtering based on location, category, month
-ability to mark tender and add to favorities so that he can receive all information regarding that tender

Tender Notification Settings
-where we member can select to receive which tender notification to receive based on location, category or all tenders
